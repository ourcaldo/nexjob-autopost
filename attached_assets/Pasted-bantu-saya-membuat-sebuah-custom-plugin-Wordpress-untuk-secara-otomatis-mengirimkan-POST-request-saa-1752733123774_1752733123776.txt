Perhatikan berikut ini:
1. Untuk tags, dapatkan data dari tags di post tersebut (tag-loker).
2. Hapus image di request POST, jadi tidak perlu send field image
3. Content ganti ke:
Lowongan kerja di kota nexjob_lokasi_kota, cek selengkapnya di {link post}. 

#hashtag

4. nexjob_lokasi_kota merupakan custom fields di cpt lowongan-kerja (I made with jetengine)
5. Hashtag tambahkan semua tags di tag-loker
6. Untuk {link-post} akan seperti cms.nexjob.tech/*/*. Gantikan dengan nexjob.tech/*/*

Saya juga ingin ada log record yang bisa saya lihat dan hapus dan auto hapus sesuai dengan interval.

Untuk setting, buat seperti fields, untuk link API dan authorization header sehingga jika perlu saya bisa mengganti nantinya

Oh ya, keep in mind untuk post yang di kirim ke API itu bukan hanya post yang di publish secara manual (dengan mengklik tombol Publish di WP), tapi juga Post yang datang dari API REST wordpress atau 3rd party plugin (case saat ini saya menggunakan auto post dari luar WP, saya kirim ke webhook plugin dan plugin membuat post baru yang langsung publish).
